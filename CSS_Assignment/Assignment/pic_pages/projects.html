Animations in Astronomy

Animations in the field of astrophysics play a very crucial role in that they enthrall the general audiences and help promote the fieldX while also providing researchers with a way to visualise complex theories that are otherwise either simplified or discarded. As the processing power of computers has increased manifold over the yearsX it has become easier to mimic astrophysical systems without too much manual effort. 8n this projectX we will create animations which could be automated using python that would allow us to recreate realistic astrophysical scenarios.
Objectives for this project :
  • Learn basics of an industrial standard animation software Tool (Blender)
  • Procedurally generate realistic physics animation/simulation
  • Learn how to read astronomy data files using Python
  • Use Python to simulate simplified condition (in matplotlib)
  • Integrate python interface with blender
  • Optimise the python code
